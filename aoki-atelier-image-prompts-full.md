# Aoki Atelier — 画像生成プロンプト依頼コマンド集（完全版）

## 共通指示（全プロンプトの冒頭に付ける）

```
以下のプロンプトでジュエリーブランドサイト用の画像を生成してください。
共通ルール:
- 背景: ダーク（#0a0a0a〜#1a1a1a の黒〜ほぼ黒）
- ライティング: ソフトな自然光風、ハイライト控えめ、マットな金属質感
- 構図: 被写体を中央〜やや偏心に配置、十分な余白
- 人物が映る場合: 手元のみ（顔は映さない）
- 解像度: 1024x1024
- スタイル: ハイエンドジュエリーブランドの商品撮影風、写実的
```
